### 🚀 Plano Definitivo: IronTracks VIP Ecosystem

Com base na sua diretriz, o **Active Workout** (ferramentas de treino) permanecerá **100% Free**. Isso cria um funil poderoso: o usuário vicia no treino e paga para ter a inteligência (IA) e os dados (Nutrição/Analytics).

Aqui está a estrutura final detalhada e o plano de execução.

### 🎨 2. Remodelagem Visual (UX/UI)

*   **Seções:**
    1.  **Status Bar:** "Você é PRO" • "IA Restante: 32/40".
    2.  **Quick Actions (Atalhos):**
        *   "💬 Falar com Coach" (Abre o chat).

    3.  **Vitrine de Features (Upsell):**
        *   Para Free/Start: Cards bloqueados 🔒 ("Desbloqueie o Chef IA no Elite").

#### **B. Nova Tela de Planos (Marketplace)**
*   **Design:** Tabela Comparativa (Pricing Table) inspirada em SaaS (ex: Spotify/Strava).
*   **Detalhes:**
    *   **Switch:** "Mensal" vs "Anual (2 meses grátis)".
    *   **Comparativo:** Lista linha a linha com ✅ e ❌ para deixar claro o que ganha.
    *   **Destaque:** Tag "Mais Popular" no plano PRO.

### 🛠️ 3. Plano de Execução Técnica (Passo a Passo)

1.  **Backend (Infraestrutura de Limites):**
    *   Criar tabela/coluna `vip_limits` no Supabase (ou usar jsonb em `app_plans`) para parametrizar: `{ "chat_daily": 10, "wizard_weekly": 1, "history_days": 60 }`.
    *   Implementar `checkVipLimit(userId, feature)` nas APIs:
        *   `/api/ai/coach-chat` (Chat)
        *   `/api/ai/workout-wizard` (Wizard)
        *   `/api/ai/post-workout-insights` (Relatório IA)

2.  **Frontend (Telas Novas):**
    *   Recriar `src/app/marketplace/page.tsx` com a nova Tabela Comparativa.
    *   Recriar `src/components/VipHub.js` com o novo layout de Dashboard.

3.  **Frontend (Bloqueios/Gates):**
    *   **CoachChatModal:** Remover lógica local e usar resposta da API (403 Forbidden) para mostrar modal de upgrade.
    *   **NutritionMixer:** Esconder gráficos de macros se plano < PRO.
    *   **HistoryList:** Filtrar query por data baseada no plano.

**Autoriza o início? Vou começar pela infraestrutura de limites no Backend e depois avançar para as telas.**
