Vou atualizar a configuração do Capacitor para usar a URL oficial que você forneceu: `https://app-iron-tracks.vercel.app/`.

Isso deve resolver o erro de "redirect_uri_mismatch" do Google, pois essa URL provavelmente já está autorizada no seu painel.

1.  Editar o arquivo `capacitor.config.ts`.
