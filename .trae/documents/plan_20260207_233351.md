# Implementar Login por E-mail/Senha com Persistência

Sim, o login por e-mail e senha mantém a sessão ativa por longo prazo (já configurado para 1 ano no projeto), evitando o redirecionamento constante do Google.

## 1. Atualizar `LoginScreen.js`
- **Novo Formulário:** Adicionar campos de E-mail e Senha.
- **Alternância:** Botões para escolher entre "Google" ou "E-mail".
- **Cadastro:** Botão "Criar Conta" para novos usuários.
- **Recuperação:** Link "Esqueci a senha" (essencial para quem migra do Google).

## 2. Unificação de Contas
- Se o usuário já usa Google, ele usará o "Esqueci a senha" com o mesmo e-mail. Isso enviará um link para definir uma senha, permitindo que ele acesse **a mesma conta** (com todo o histórico) tanto pelo Google quanto por senha.

## 3. Persistência
- O sistema já está configurado para manter o login por 1 ano. Com e-mail/senha, isso é transparente (sem popups ou redirecionamentos externos), garantindo a sensação de "sempre logado".

Vou aplicar as mudanças no `src/components/LoginScreen.js`.
