Vou corrigir o comportamento do botão "BAIXAR / COMPARTILHAR" no `StoryComposer.tsx`.

### Problema
O erro `The request is not allowed by the user agent...` ocorre porque o navegador bloqueou a tentativa de compartilhamento nativo (`navigator.share`), mas o código atual trata isso como um erro fatal e exibe o alerta vermelho, em vez de tentar fazer o download do arquivo como alternativa.

### Solução
Vou alterar a função `shareImage` para implementar um **fallback automático**:
1. Tentar usar o compartilhamento nativo (`navigator.share`).
2. Se o usuário cancelar (`AbortError`), não fazer nada (comportamento correto).
3. Se o compartilhamento falhar por qualquer outro motivo (como o erro de permissão que você viu), **baixar a imagem automaticamente** (`downloadBlob`) em vez de mostrar o erro.

Isso garante que, se o compartilhamento falhar, você ainda consegue salvar a imagem.

### Alterações
- Arquivo: `src/components/StoryComposer.tsx`
- Função: `shareImage`
- Lógica: Envolver o `navigator.share` em um `try/catch` específico para permitir o fallback para download.
